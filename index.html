Can you help me implement a dark theme for my website?
CF

Sure! Here's a simple implementation of a dark theme toggle using CSS variables:

// CSS Variables
:root {
  --bg-color: #ffffff;
  --text-color: #333333;
}

body.dark-theme {
  --bg-color: #1a1a1a;
  --text-color: #f0f0f0;
}

// JavaScript Toggle
const themeToggle = document.getElementById('themeToggle');
themeToggle.addEventListener('click', () => {
  document.body.classList.toggle('dark-theme');
});
